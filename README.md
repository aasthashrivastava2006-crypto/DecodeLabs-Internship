# DecodeLabs Internship

This repository contains the Java projects completed during my internship at **DecodeLabs**. These projects demonstrate fundamental Java programming concepts, object-oriented programming (OOP), user interaction, input validation, and problem-solving skills.

## 📌 Projects

### 1. 🎓 Student Grade Calculator
A Java console application that calculates the total marks, average percentage, and final grade based on marks entered for multiple subjects. The program validates user input and displays the student's pass/fail status. :contentReference[oaicite:0]{index=0}

#### Features
- Enter marks for multiple subjects
- Input validation (0–100)
- Calculate total marks
- Calculate average percentage
- Assign grades (A–F)
- Display Pass/Fail status

#### Technologies Used
- Java
- Scanner Class
- Loops
- Conditional Statements

### 2. 🎮 Number Guessing Game
A console-based Java game where the player has to guess a randomly generated number between 1 and 100 within a limited number of attempts. The game provides hints, tracks scores, and allows multiple rounds. :contentReference[oaicite:1]{index=1}

#### Features
- Random number generation
- Maximum attempt limit
- Too High / Too Low hints
- Score calculation
- Play multiple rounds
- Final score display

#### Technologies Used
- Java
- Random Class
- Scanner Class
- Loops
- Conditional Statements

### 3. 🏧 ATM Interface
A Java console-based ATM simulation that allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money through a menu-driven interface. The project uses object-oriented programming by separating the Bank Account and ATM functionalities. :contentReference[oaicite:2]{index=2}

#### Features
- Check account balance
- Deposit money
- Withdraw money
- Input validation
- Menu-driven interface
- Object-Oriented Design

#### Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Exception Handling

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/<your-username>/DecodeLabs-Internship.git
```

2. Navigate to the project folder

```bash
cd DecodeLabs-Internship
```

3. Compile the Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```

Replace **FileName** with:

- `StudentGradeCalculator`
- `NumberGame`
- `AtmInterface`
